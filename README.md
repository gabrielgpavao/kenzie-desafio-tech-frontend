## 1. Início Rápido

### 1.1. Instalando Dependências

Clone o projeto em sua máquina e instale as dependências com o comando:

Caso use npm
```bash
npm install
```

Caso use yarn
```bash
yarn
```

### 1.2. Inicialização do Host

Inicie o seu host localmente com o comando:

Caso use npm
```bash
npm run dev
```

Caso use yarn
```bash
yarn dev
```
